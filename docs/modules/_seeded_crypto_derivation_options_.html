SeededCryptoDerivationOptions

SeededCryptoDerivationOptions<TYPE, HASH_FUNCTION>: HashFunctionOptions<HASH_FUNCTION>

The DerivationOptions used by the Seeded Crypto library and which implement the portion of the JSON Derivation Options format. that are interpreted by this library.

(Other options may appear in layers above that library.)

Type parameters

  • TYPE: DerivableObjectName

  • HASH_FUNCTION: HashFunction
